HTML Semántico

HTML semántico es el uso de las etiquetas HTML para estructurar el contenido de una página web de manera que sea más fácil de entender para los humanos y los robots de los buscadores.

  • header: encabezado de la página
  • nav: navegación de la página
  • main: contenido principal de la página
  • article: artículo de página
  • section: sección de la página
  • footer: pie de página
